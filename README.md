# NitroRelay

スマホが爆弾になる、飲み会用リレーゲーム

## 概要

NitroRelay は、飲み会やパーティーで 1 台の iPhone を回しながら遊ぶ「爆弾リレー」ゲームです。
スマホの揺れを検知し、一定以上揺れると「ドカーン！」と爆発します。

## 開発

iOS 16.0 以降の iPhone を対象とした SwiftUI アプリです。

### 要件

- Xcode 14.0 以降
- iOS 16.0 以降
- Swift 5.7 以降

### ビルド

```bash
open NitroRelay.xcodeproj
```

Xcodeでプロジェクトを開き、ビルドして実行してください。
